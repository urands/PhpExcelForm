## Installation
Now PhpExcelForm only dev version. You must add PhpExcelForm repository manualy in your composer.json file:
```
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/urands/phpexcelform"
        }
    ]
```
Aftet that:
```
	composer require urands/phpexcelform:dev-master
```
